DevOps

Cloud Computing & DevOps

📅 Semester 6 • 🎓 5 ECTS Credits • 👨‍🏫 Prof. Kelsey Hightower

Course Overview

This course bridges the gap between software development and IT operations. Students will gain practical experience in modern cloud architectures, deploying applications using AWS and Azure, managing containers with Docker and Kubernetes, and automating workflows through Continuous Integration and Continuous Deployment (CI/CD) pipelines.

Learning Objectives

  • Understand core cloud computing models (IaaS, PaaS, SaaS) and navigate AWS/Azure ecosystems.
  • Containerize applications using Docker and orchestrate deployments at scale with Kubernetes.
  • Automate infrastructure provisioning using Infrastructure as Code (IaC) tools like Terraform or Ansible.
  • Design, implement, and maintain robust CI/CD pipelines (e.g., using GitLab CI, GitHub Actions, or Jenkins).

Prerequisites

Familiarity with the Linux command line, basic networking concepts, and software development lifecycles (Git version control) are highly recommended.
