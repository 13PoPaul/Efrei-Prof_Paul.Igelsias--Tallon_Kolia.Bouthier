Data Science

Machine Learning Foundations

📅 Semester 4 • 🎓 5 ECTS Credits • 👨‍🏫 Prof. Alan Turing

Course Overview

This course introduces the fundamental concepts of Machine Learning. Students will explore both supervised and unsupervised learning paradigms, mastering algorithms for regression, classification, and clustering, before advancing to an introduction to the architecture of artificial neural networks.

Learning Objectives

  • Understand the differences between supervised and unsupervised learning approaches.
  • Implement and evaluate predictive models using linear/logistic regression and classification trees.
  • Apply clustering algorithms (like K-Means) to discover hidden patterns within unlabeled data.
  • Grasp the foundational mathematics and architecture behind basic Neural Networks.

Prerequisites

A solid foundation in linear algebra, statistics, calculus, and proficiency in Python programming (especially libraries like NumPy and Pandas) is highly recommended.
