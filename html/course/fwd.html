Efrei Prof - Course Details

Web Development

Full-Stack Web Development

📅 Semester 2 • 🎓 5 ECTS Credits • 👨‍🏫 Prof. Sarah Lee

Course Overview

This course focuses on building scalable and modern web applications from scratch. Students will learn how to design dynamic front-end interfaces using React and modern CSS frameworks, while developing robust back-end services with Node.js and REST APIs.

Learning Objectives

  • Master modern front-end development using React and component-based architecture.
  • Build and deploy scalable back-end environments using Node.js and Express.
  • Design, implement, and consume RESTful APIs for seamless client-server communication.
  • Utilize modern CSS frameworks (like Tailwind CSS or Bootstrap) to create responsive, accessible user interfaces.

Prerequisites

A solid foundation in HTML5, CSS3, and modern vanilla JavaScript (ES6+) is highly recommended before taking this course.
