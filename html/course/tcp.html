Networking

TCP/IP Networks

📅 Semester 2 • 🎓 5 ECTS Credits • 👨‍🏫 Prof. Vint Cerf

Course Overview

This course provides a comprehensive introduction to computer networking principles. Students will explore the OSI and TCP/IP models, master IPv4/IPv6 subnetting, implement routing protocols, and learn how to capture and analyze network packets to troubleshoot real-world connectivity issues.

Learning Objectives

  • Understand the 7 layers of the OSI model and how they map to the TCP/IP protocol suite.
  • Master IP addressing, CIDR notation, and advanced subnetting techniques.
  • Configure and troubleshoot routing protocols (such as OSPF and BGP) and switching concepts.
  • Perform network packet analysis and diagnose network anomalies using Wireshark.

Prerequisites

Basic knowledge of computer hardware, operating systems (Windows/Linux), and a general understanding of IT infrastructure are recommended.
