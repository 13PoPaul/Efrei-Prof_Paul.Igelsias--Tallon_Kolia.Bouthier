Cybersecurity

Network Security & Ethical Hacking

📅 Semester 5 • 🎓 5 ECTS Credits • 👨‍🏫 Prof. Elliot Alderson

Course Overview

This course introduces the methodologies of ethical hacking and network security. Students will learn how to think like an attacker to proactively identify, exploit, and patch vulnerabilities in IT infrastructures using industry-standard penetration testing techniques and tools.

Learning Objectives

  • Understand and apply the phases of penetration testing: Reconnaissance, Scanning, Exploitation, and Post-Exploitation.
  • Conduct comprehensive vulnerability assessments on networks and web applications.
  • Master industry-standard security tools such as Nmap, Wireshark, Metasploit, and Burp Suite.
  • Design and implement secure infrastructure architectures to defend against common cyber attacks.

Prerequisites

A strong understanding of networking protocols (TCP/IP, DNS, HTTP), Linux command-line proficiency, and basic scripting skills (Python or Bash) are highly recommended.
