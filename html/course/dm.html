Efrei Prof - Course Details

Data Engineering

Databases Management

📅 Semester 3 • 🎓 5 ECTS Credits • 👨‍🏫 Prof. Ada Lovelace

Course Overview

This course covers the core concepts of data storage, modeling, and retrieval. Students will explore relational algebra, master advanced SQL techniques using PostgreSQL, and dive into NoSQL document-oriented databases using MongoDB. Additionally, the course emphasizes database optimization and performance tuning.

Learning Objectives

  • Understand relational algebra, database design, and normalization (1NF, 2NF, 3NF).
  • Write complex and optimized SQL queries, including JOINs, subqueries, and window functions in PostgreSQL.
  • Design, query, and manage non-relational (NoSQL) databases using MongoDB.
  • Implement database optimization techniques such as indexing, query execution plan analysis, and caching.

Prerequisites

Basic knowledge of programming logic and familiarity with fundamental data structures are recommended.
