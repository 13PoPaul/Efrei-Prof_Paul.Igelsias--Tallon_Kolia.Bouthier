Efrei Prof - Course Details

Data Engineering

Big Data Processing

📅 Semester 5 • 🎓 5 ECTS Credits • 👨‍🏫 Prof. Grace Hopper

Course Overview

This course focuses on handling, processing, and analyzing massive datasets that exceed the capacities of traditional databases. Students will explore the Hadoop ecosystem, utilize Apache Spark for fast distributed computing, and learn how to build scalable, fault-tolerant data pipelines.

Learning Objectives

  • Understand the core challenges and paradigms of Big Data (Volume, Velocity, Variety, Veracity).
  • Work with the Hadoop Distributed File System (HDFS) and the MapReduce processing model.
  • Develop high-performance distributed applications using Apache Spark (RDDs, DataFrames, and Spark SQL).
  • Design and deploy automated, scalable data pipelines for batch and stream processing.

Prerequisites

A solid foundation in databases (SQL and NoSQL) and proficiency in Python or Scala programming are highly recommended.
